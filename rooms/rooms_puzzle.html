Puzzle Rooms

Puzzle Rooms contain four buttons that need to be pressed in the correct order to progress. Each button is Next to a wall, and each wall has three symbols on it.

Solving the Puzzle

The order in which the buttons need to be pressed is based on the symbols on the corresponding walls. Adding together the values for the three symbols on a wall gives the value of the associated button. Solving the puzzle requires pressing each button in order from highest value to lowest value.
